Olá Bem vindo, Desenvolvido por Matheus Mendes

Formado em Gestão de TI, pela Fatec de Itaquaquecetuba, com experiência em desenvolvimento e análises de sistemas. Durante 1 ano fui estagiário em desenvolvimento WEB, utilizando os conceitos básicos aprendidos para a manutenção de um sistema Legado em VB6, convertido para uma versão WEB, utilizando as tecnologias HTML5, CSS3, JavaScript e VB.NET, juntamente com a manipulação da base de dados utilizando o SQL Server.
Após isso assumi o cargo de Desenvolvedor FULLSTACK Júnior, realizando toda a manutenção da interface gráfica, juntamente com a correção da base de dados, otimização de consultas e geração de relatórios através das informações que o cliente solicitava ao sistema.
Possuo experiência com a homologação do sistema diretamente com o cliente, realizando as devidas manutenções e melhorias recorrentes que o sistema necessitava. Me mantenho estudando diferentes tecnologias, pois o mercado de TI exige um conhecimento bem mais amplo do que apenas sintaxe e lógica.
